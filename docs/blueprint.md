# **App Name**: Bymimi Landing

## Core Features:

- Multilingual Support: Offer content in English, French, and Arabic, with RTL support. Users can switch languages via a toggle in the navigation.
- Sticky Navigation: Implement a sticky navigation bar containing the logo, language switcher, and social media icons.
- Categorized Product Gallery: Display clothing, baby items, and accessories in a categorized product gallery with image hover effects.
- Social Connectivity: Provide stylish buttons for Instagram and TikTok, and a floating "Order via WhatsApp" button.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) for a cozy and elegant feel, inspired by the natural tones of handmade crochet.
- Background color: Off-white (#FAFAFA) to provide ample whitespace, contributing to a minimalist aesthetic.
- Accent color: Pale rose (#FFC0CB) used sparingly for highlights and calls to action, adding a gentle touch.
- Headings: 'Playfair', serif, for an elegant and fashionable headline.
- Body: 'PT Sans', sans-serif, a modern and clean font for body text.
- Utilize modern Lucide icons (e.g., a hanger for clothing, a heart for baby items, and a smartphone for pouches).
- Employ a soft minimalist theme with plenty of whitespace.
- Implement image hover effects with a subtle zoom to add interactivity.
